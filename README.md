## 📝 Deskripsi

**Digital Atelier** adalah sistem manajemen perpustakaan premium yang modern dan responsif, dibangun menggunakan **Node.js/Prisma** di sisi backend dan **React/TypeScript** di sisi frontend. Dirancang khusus untuk kebutuhan sertifikasi kompetensi (USK), sistem ini menyediakan solusi digital lengkap mulai dari manajemen koleksi buku hingga verifikasi peminjaman berbasis kode unik.

Berfokus pada estetika dan keunggulan teknis, Digital Atelier menghadirkan transisi yang halus, validasi real-time, dan antarmuka bersih yang menjembatani kenyamanan perpustakaan konvensional dengan fungsionalitas modern.

## ✨ Fitur Utama

- **🎨 UX/UI Premium**: Desain modern dengan estetika bersih, mendukung Dark/Light mode dan animasi mikro.
- **🔐 Sistem RBAC Terproteksi**: Alur kerja dan dashboard yang berbeda untuk **Petugas (Admin)** dan **Pengunjung (Member)**.
- **📚 Katalog Pintar**: Pencarian canggih, filter kategori, dan pemantauan stok buku secara real-time.
- **🔄 Mesin Sirkulasi**: Siklus peminjaman dan pengembalian otomatis dengan sistem verifikasi kode unik.
- **📊 Kendali Admin**: Manajemen CRUD lengkap untuk buku, kategori, dan pengguna disertai statistik transaksi harian.
- **📱 Responsivitas Maksimal**: Dioptimalkan secara mulus untuk perangkat mobile, tablet, hingga desktop.

## ✨ Teknologi yang Digunakan

<details><summary>Kekuatan Fullstack di balik Digital Atelier:</summary>

### 🛠️ Backend Stack
- [Node.js](https://nodejs.org/): Runtime JavaScript sisi server yang skalabel.
- [TypeScript](https://www.typescriptlang.org/): Integritas dan keamanan kode dengan sistem typing.
- [Prisma ORM](https://www.prisma.io/): Manajemen database yang modern dan type-safe.
- [Supabase](https://supabase.com/): Database PostgreSQL dan cloud connection pooling.
- [Express](https://expressjs.com/): Framework web minimalis untuk API.

### 🎨 Frontend Stack
- [React](https://react.dev/): Library untuk membangun UI yang dinamis dan interaktif.
- [Vite](https://vitejs.dev/): Tooling frontend generasi terbaru untuk performa cepat.
- [Tailwind CSS](https://tailwindcss.com/): Framework CSS kelas atas untuk styling cepat.
- [Shadcn UI](https://ui.shadcn.com/): Komponen UI premium siap pakai.
- [Framer Motion](https://www.framer.com/motion/): Untuk animasi mikro yang elegan.

</details><br/>
